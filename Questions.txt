Y axeln represenenterar frekvensen som en viss random variabel förekommer. Vid N = 1 går det att se att X är exponentialfördelad. Vid stora värden för N (> 1000) Så sker ingen märkbarn förändering på distrubionen när N ökas med ett steg. Dock så närmar sig distrubionen en normalfördelning ju större N är. Detta beror på den centrala gränsärde satsen som säger att summan av oberoende variabler går mot en normalfördelning när n -> infinity.
